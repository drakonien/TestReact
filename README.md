# ToDo Vite React (TypeScript)

Simple ToDo app created with Vite + React + TypeScript. Includes unit tests using Vitest and Testing Library.

Quick start

## 1. Install dependencies

```bash
npm install
```

## 2. Run the dev server

```bash
npm run dev
```

## 3. Run tests

```bash
npm run test
```

## Build

```bash
npm run build
npm run preview
```

Features

- Add new items (duplicate prevention)
- Toggle completed state
- Delete items

Notes

- To run tests locally, first run `npm install` to install dependencies.

## Verify packages versions

```bash
npx npm-check-updates
```
